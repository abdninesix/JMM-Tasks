# Student Result Management System
### Laravel Internship Weekly Task

A comprehensive Laravel application designed to demonstrate core PHP and Laravel concepts, including conditional logic, loops, modular functions, array manipulation, and database management.

## Features

This project is divided into 5 learning tasks and a final capstone dashboard:

*   **Task 1: Conditional Statements**
    *   Automated grading system (A+, A, B, etc.) based on marks.
    *   Pass/Fail status determination.
    *   Scholarship eligibility check (Grade A+ and Attendance > 90%).
    *   Attendance warning system for students below 75%.
*   **Task 2: Loops & Iterations**
    *   Dynamic student listing using `@forelse`.
    *   Calculation of total class marks and averages.
    *   Demonstration of `for` and `while` loop logic.
*   **Task 3: Modular Functions**
    *   Encapsulated logic using private controller methods.
    *   Math utilities (Square calculation, BMI calculator).
    *   Formatting utilities (Currency and Percentage formatting).
*   **Task 4: Data Manipulation (Arrays & Collections)**
    *   Search functionality by Name or Roll Number.
    *   Sorting logic (Descending order by marks).
    *   Identifying class toppers and summary statistics.
*   **Task 5: String Handling**
    *   Dynamic username generation using `explode()`.
    *   Professional name formatting using `ucwords()` and `strtolower()`.
    *   Character counting and excerpt generation.

## Technologies Used

*   **Framework:** Laravel 13
*   **Frontend:** Tailwind CSS & Blade Templating Engine
*   **Database:** MySQL
*   **Language:** PHP

## Installation & Setup

1.  Clone the repository
2.  Setup the database for MySQL
